Q: Command seems stuck on "npm install"
A: This is normal - npm install takes 3-5 minutes with 1,200+ packages
   Just wait, don't interrupt

Q: Getting permission denied errors
A: Verify SSH key is setup correctly:
   ssh-copy-id ubuntu@lmsetjen.id

Q: Services won't start after deployment
A: Check logs:
   ssh ubuntu@lmsetjen.id "docker-compose logs"

Q: Certificate QR codes still not working after deployment
A: Restart backend service:
   ssh ubuntu@lmsetjen.id "cd /home/ubuntu/LMSetjen-DPD-RI && docker-compose restart backend"

Q: Need to rollback
A: Rollback deployment:
   ssh ubuntu@lmsetjen.id << 'EOF'
   cd /home/ubuntu/LMSetjen-DPD-RI
   git reset --hard HEAD~1
   npm install
   npm run build
   docker-compose restart
   EOF
