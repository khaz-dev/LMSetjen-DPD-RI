🔧 ESSENTIAL COMMANDS REFERENCE
═════════════════════════════════════════════════════════════════════════════════

Deployment:
  docker-compose up -d                          Start all services
  docker-compose down                           Stop all services
  docker-compose restart                        Restart services
  docker-compose down -v                        ⚠️  Delete everything

Monitoring:
  docker ps                                     List running containers
  docker-compose logs -f backend                View backend logs
  docker-compose logs -f                        View all logs
  docker stats                                  CPU/memory usage

Management:
  docker exec -it lms_backend bash              Enter backend container
  docker exec -it lms_postgres psql -U lms_user Backend database shell
  docker-compose build --no-cache               Rebuild all images
  docker system df                              Check Docker disk usage

Database:
  docker exec -i lms_postgres pg_dump ...       Backup database
  docker exec -i lms_postgres psql ...          Restore database
  docker exec lms_backend python manage.py ...  Run Django commands

User Management:
  docker exec -it lms_backend python manage.py createsuperuser
                                                Create admin user
