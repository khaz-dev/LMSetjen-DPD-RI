📋 ISSUE SUMMARY

❌ Instructor QA page showing 0 Questions for all courses
✅ Database confirmed to have 1 actual question in Course ID 5
❓ Why? Backend serializers were incomplete + frontend logic was broken

🔍 ROOT CAUSE ANALYSIS
─────────────────────────────────────────────────────────────────────────────────

BACKEND PROBLEMS:
  1. CourseSerializer → missing "qa_count" field
  2. Question_AnswerSerializer → "course" was just integer ID, not object

FRONTEND PROBLEMS:
  1. Trying to filter: q.course?.id === course.id
  2. BUT q.course was integer 5, not object {id: 5, ...}
  3. So: 5?.id = undefined, undefined === 5 = FALSE
  4. Made N+1 API calls (inefficient)
