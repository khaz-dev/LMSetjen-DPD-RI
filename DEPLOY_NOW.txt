Solution:
  npm install --force
  npm install --prefer-offline --no-audit

Problem: Build fails
Solution:
  cd frontend
  rm -rf dist/
  npm run build

Problem: Services won't start
Solution:
  docker-compose logs -f
  docker-compose restart
  docker-compose up -d

Problem: Certificate QR codes still not working
Solution:
  cd /home/ubuntu/LMSetjen-DPD-RI
  docker-compose restart backend
  Wait 30 seconds
  Test again

Full Rollback (if needed):
  cd /home/ubuntu/LMSetjen-DPD-RI
  git reset --hard HEAD~1
  npm install
  npm run build
  docker-compose restart
