# Quick Reference: Student Page Spacing Control\n\n## Where to Change Spacing\n\n**File**: `frontend/src/index.css` (Line ~177-182)\n\n## Current Values\n\n```css\n:root {\n  --navbar-height: 70px;\n  --page-top-spacing: 3rem;         /* ← CHANGE THIS for gap from navbar */\n  --page-bottom-spacing: 3rem;      /* ← CHANGE THIS for bottom padding */\n  --header-card-gap: 1.5rem;        /* ← CHANGE THIS for header card gap */\n}\n```\n\n## What Each Variable Controls\n\n| Variable | Default | Controls |\n|----------|---------|----------|\n| `--navbar-height` | 70px | Navbar height (used in calc) |\n| `--page-top-spacing` | 3rem | Gap between navbar and page content |\n| `--page-bottom-spacing` | 3rem | Bottom padding of all student pages |\n| `--header-card-gap` | 1.5rem | Space below header card |\n\n## Affected Pages\n\nChanging these variables updates spacing on:\n- Dashboard\n- Courses\n- Profile\n- Wishlist  \n- Certificates\n- Testimonials\n- Q&A\n- Course Details\n- Change Password\n\n## Quick Changes\n\n### Increase top gap to 4rem\n```css\n--page-top-spacing: 4rem;\n```\n\n### Reduce bottom padding to 2rem\n```css\n--page-bottom-spacing: 2rem;\n```\n\n### Tighter header card gap\n```css\n--header-card-gap: 1rem;\n```\n\n### Larger header card gap\n```css\n--header-card-gap: 2rem;\n```\n\n## How It Works\n\nCSS variables replace hardcoded values in all student page CSS files:\n- `calc(70px + 3rem)` → `calc(var(--navbar-height) + var(--page-top-spacing))`\n- `1.5rem` → `var(--header-card-gap)`\n- `3rem` → `var(--page-bottom-spacing)`\n\n**Changes apply immediately after saving!** (hot reload on dev server)\n\n## Dev Server\n\nRun: `npm run dev` in `frontend/` folder\nVisit: `http://localhost:5174/student/dashboard/` (or 5175 if port busy)\n\n## Files Using These Variables\n\n- Dashboard.css\n- Courses.css\n- Profile.css\n- Wishlist.css\n- SertifikatKursus.css\n- Testimonials.css\n- ChangePassword.css\n- CourseDetail.css\n- QA.css\n- QADetail.css\n- Header.css (Partials)\n\n---\n\n📌 **Pro Tip**: These are CSS custom properties (CSS variables) - they cascade from `:root` to all child elements. Change once, update everywhere!\n
