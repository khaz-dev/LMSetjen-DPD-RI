💡 KEY TECHNICAL DETAILS
═════════════════════════════════════════════════════════════════════════════════

  Bug Pattern:
    Backend returns:  https://host/media/course-file/uuid.png
    Frontend bug:     transforms to /api/media/course-file/uuid.png (404)
    
  Fix Pattern:
    Check for full URL FIRST
    └─ if (startsWith('http://') || startsWith('https://')) return url
    └─ THEN handle relative paths

  Files Modified:
    ├─ courseUtils.js (moved http check before path extraction)
    └─ Dashboard.jsx (applied same fix)
