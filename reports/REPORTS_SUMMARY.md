## 🖨️ HOW TO CONVERT TO PDF

### Method 1: Browser (EASIEST - Recommended)
1. Open any HTML file in Chrome/Firefox/Edge
2. Press **Ctrl + P** (or Cmd + P on Mac)
3. Select **"Save as PDF"** as printer
4. Click **Save**
5. ✅ Done! PDF generated with perfect formatting

**Browser Settings:**
- Paper: A4
- Margins: Default
- Background Graphics: ✓ ON
- Headers/Footers: OFF

### Method 2: Python Script (Batch Conversion)
```bash
# Install requirements
pip install pdfkit

# Install wkhtmltopdf
# Windows: Download from https://wkhtmltopdf.org/downloads.html

# Run converter
cd reports
python convert_reports_to_pdf.py
```

### Method 3: Microsoft Word
1. Open HTML in Word
2. File → Save As → PDF
3. Save
