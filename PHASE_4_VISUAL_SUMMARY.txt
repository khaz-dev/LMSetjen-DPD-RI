    ↓

Example 1: Access Role Data

import { useRoles } from '../utils/useRoles';

function MyComponent() {
  const { availableRoles, currentRole } = useRoles();
  
  return (
    <div>
      Current Role: {currentRole}
      Available: {availableRoles.join(', ')}
    </div>
  );
}


Example 2: Use Role Utilities
─────────────────────────────

import { switchRole, hasRole, isCurrentRole } from '../utils/roleUtils';

async function handleRoleChange(newRole) {
  if (hasRole(newRole)) {  // Check if user has the role
    const result = await switchRole(newRole);
    if (result.success) {
      console.log('Switched to:', result.current_role);
    }
  }
}

if (isCurrentRole('admin')) {
  // Show admin features
}

CONTEXT STRUCTURE
=================

RolesContext provides:
{
  availableRoles: string[],      // ['student', 'teacher', 'admin']
  currentRole: string,            // 'teacher'
  rolesLoading: boolean,          // false
  fetchAvailableRoles: function   // () => Promise
}

Fallback (if context not provided):
{
  availableRoles: [],
  currentRole: null,
  rolesLoading: false,
  fetchAvailableRoles: () => {}
}


API INTEGRATION
===============

Endpoint: GET /api/v1/auth/available-roles/
─────────────────────────────────────────────

Request:
  Headers: Authorization: Bearer {jwt_token}

Response (200 OK):
  {
    "success": true,
    "available_roles": ["student", "teacher", "admin"],
    "current_role": "teacher",
    "user_id": 1,
    "email": "user@example.com",
    "full_name": "John Doe",
    "has_multiple_roles": true,
    "timestamp": "2025-01-25T..."
  }

Error Handling:
  - Network error → Fallback to user data if available
  - 401 Unauthorized → User not logged in
  - 400 Bad Request → Invalid request
  - Server error → Log error, show fallback


PERFORMANCE OPTIMIZATION
========================

Memoization:
✓ rolesContextValue = useMemo(
    () => ({ availableRoles, currentRole, rolesLoading, fetchAvailableRoles }),
    [availableRoles, currentRole, rolesLoading]
  )

Benefits:
  ✓ Context doesn't update unnecessarily
  ✓ Components don't re-render on every change
  ✓ Prevents infinite loops
  ✓ Better performance overall
